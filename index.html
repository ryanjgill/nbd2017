Getting started with the Tessel

Install Tessel CLI

npm install -g t2-cli

Note: If you get the common npm EACCES error, correct it by fixing npm permissions and run the command again.

https://docs.npmjs.com/getting-started/fixing-npm-permissions

Run a file

t2 run index.js
t2 run name_of_file

Copy the file and its dependencies into Tessel's RAM & run immediately. Use this during development of your device application.

Flag Description
--lan deploy over LAN connection
--usb deploy over USB connection
--slim copy only files needed by the program to run
--full copy all the files in the project directory

Push a file

t2 push index.js
t2 push name_of_file

Copy the file and its dependencies into Tessel's Flash memory & run immediately. Once deployed with push command, the device application will automatically run every time the Tessel restarts.

Flag Description
--lan deploy over LAN connection
--usb deploy over USB connection
--slim copy only files needed by the program to run
--full copy all the files in the project directory

Erase Files

t2 erase
t2 erase

Erase any code previously pushed to the Tessel using the t2 push command.

Manual, live control

var five = require("johnny-five");
var Tessel = require("tessel-io");
var board = new five.Board({
  io: new Tessel()
});

board.on("ready", function() {
  // Must use pins 5 or 6 for PWM
  var left = new five.Servo({pin: "A5"});
  var right = new five.Servo({pin: "A6", invert: true});
  var servos = new five.Servos([left, right]);

  var commands = {
    left: function() { left.cw();right.ccw(); },
    right: function() { left.ccw();right.cw(); },
    fwd: function(speed) { servos.cw(speed); },
    rev: function(speed) { servos.ccw(speed); },
    stop: function() { servos.stop(); }
  };

  this.repl.inject(commands);
});

PWM

Pulse Width Modulation, or PWM, is a technique for getting analog results with digital means.
Digital control is used to create a square wave, a signal switched between on and off.
- Arduino.cc

Convert 10-bit Analog Input to 8-bit PWM Output

var five = require("johnny-five");
var Tessel = require("tessel-io");
var board = new five.Board({
  io: new Tessel()
});

board.on("ready", function() {
  var led = new five.Led(11);
  var rotary = new five.Sensor("B0");

  rotary.on("change", function() {
    // Analog sensors produce a 10-bit value,
    // but led brightness is an 8-bit PWM value.
    // Convert by shifting the value's bits
    // two places to the right.
    led.brightness(this.value >> 2);
  });
});
